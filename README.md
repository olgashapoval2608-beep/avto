# SOVBEZ AUTO IMPORT — UX/UI кейс

Статичний сайт: інтерактивний прототип + дизайн-система.

- `index.html` — прототип (головна, каталог, авто, збережені, заявка, кабінет)
- `design-system.html` — дизайн-система та мобільні екрани

## Деплой на Vercel
1. Завантажте ці файли в корінь репозиторію `avto` (GitHub → Add file → Upload files).
2. На vercel.com: **Add New → Project → Import** репозиторій `avto`.
3. Framework Preset: **Other**, нічого не змінюйте → **Deploy**.

Жодного білду не потрібно — це чистий статичний сайт.
